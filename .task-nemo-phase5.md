Create the audio and UI files for Cosmic Invaders. Working directory: /home/carlos/Projects/cosmic-invaders/

1. src/audio/AudioManager.js - Web Audio API manager with oscillator-based retro sound effects. No external audio files needed. Methods:
   - init() creates AudioContext
   - playShoot() - short high-pitched blip
   - playExplosion() - noise burst with decay
   - playPowerUp() - ascending arpeggio
   - playHit() - low thud
   - playGameOver() - descending tone
   - playWaveStart() - fanfare
   Each sound uses OscillatorNode and GainNode with short envelopes. Export default class.

2. src/ui/HUD.js - HTML overlay for game info. Creates a div overlay on top of the canvas with:
   - Score display (top left)
   - High score (top center)
   - Lives display (top right, show ship icons as text)
   - Wave number (top center below high score)
   - update(score, lives, wave, highScore) method refreshes the display
   - show()/hide() methods
   Style with inline styles: monospace font, white text, text-shadow for glow. Export default.

3. src/ui/Menu.js - Title screen and game over screen. Creates overlay divs:
   - showTitle() - "COSMIC INVADERS" title with retro styling, "Press ENTER to Start", animated blinking
   - showGameOver(score, highScore) - "GAME OVER" with final score, "Press ENTER to Restart"
   - showPaused() - "PAUSED" overlay
   - hide() removes all overlays
   Style with CSS: centered text, retro arcade font look using monospace, neon glow text-shadow. Export default.

4. src/ui/styles.css - Base styles:
   - body: margin 0, overflow hidden, background black
   - canvas: display block
   - .hud, .menu classes with position absolute, pointer-events none
   - Retro neon text glow animation using keyframes
