Create all the enemy and gameplay files for the Cosmic Invaders game. Read src/utils/Config.js and src/game/Player.js first to understand the patterns.

Create these files:

1. src/game/Invader.js - Enemy class with Three.js mesh. Different colors per type: basic=0x00ff00, tough=0xffff00, elite=0xff0000. Hit points: basic=1, tough=2, elite=3. Properties: position, type, hp, alive. Uses BoxGeometry for the body.

2. src/game/InvaderGrid.js - Manages the 5x11 grid of Invaders using CONFIG values. Handles formation movement: move right until edge, drop down, reverse direction. Speed increases as invaders die. Random enemy shooting based on FIRE_CHANCE.

3. src/game/Bullet.js - Projectile class with small CylinderGeometry mesh. Has velocity, owner (player or enemy), and update method that moves it each frame. Color: player=0x00ffff, enemy=0xff4444.

4. src/game/Collision.js - Static collision detection. checkAABB(a,b) compares bounding boxes. Methods for bullet-vs-invader, bullet-vs-player, bullet-vs-barrier checks that return hit results.

5. src/game/Barrier.js - Destructible shield made of small box meshes arranged in an arch shape. Individual blocks can be destroyed when hit. Uses CONFIG.BARRIER values for dimensions.

6. src/game/PowerUp.js - Drops from killed enemies with CONFIG.POWERUPS.CHANCE probability. Falls downward, collected on player contact. Types: speed, multishot, shield. Has duration timer from CONFIG.

Use ES modules with export default for each class. Import THREE from 'three' and CONFIG from '../utils/Config.js'.
