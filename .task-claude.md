Create all the graphics and effects files for the Cosmic Invaders game. This is a Space Invaders-style game using Three.js with WebGPU. Read the existing src/graphics/Renderer.js first.

Create these files:

1. src/graphics/ShaderLib.js - WGSL shader library with:
   - Glow shader (pulsing emissive effect for enemies and player)
   - Shield energy shader (animated hex pattern for barriers)
   - Dissolve shader (for enemy death animation - fragments break apart)
   Export as named exports.

2. src/graphics/ParticleSystem.js - GPU-instanced particle system using Three.js InstancedMesh. Supports:
   - Explosion effect (radial burst of particles with velocity, gravity, fade)
   - Spark effect (smaller, faster particles for bullet impacts)
   - Trail effect (for bullets)
   Pool of 500 particle instances, recycled. Each particle has position, velocity, color, life, size.

3. src/graphics/PostFX.js - Post-processing pipeline using Three.js EffectComposer:
   - UnrealBloomPass for glow effects
   - Custom CRT scanline overlay (via ShaderPass with custom shader)
   - Chromatic aberration pass
   - Vignette pass
   Chain them: render -> bloom -> CRT -> chromatic -> vignette -> screen

4. src/graphics/Background.js - Procedural starfield background:
   - 3 parallax layers of stars (near=fast, mid=medium, far=slow) using Points
   - Nebula layer using a large plane with procedural noise texture
   - Slow vertical scroll to simulate movement
   - Stars are random sizes and brightness

5. src/graphics/Models.js - Procedural 3D geometry factory (no external assets):
   - createPlayerShip() - sleek arrow/triangle shape using custom BufferGeometry with cyan emissive
   - createInvader(type) - different shapes per type: basic=squid, tough=crab, elite=octopus style using merged box geometries
   - createBullet(isPlayer) - elongated cylinder with glow
   - createBarrierBlock() - small cube
   - createPowerUp(type) - spinning diamond shape with color by type

Use ES modules. Import from 'three' and 'three/addons/postprocessing/EffectComposer.js' etc.
